### Development

```bash
# Install dependencies
npm install

# Start development server (clean, build, serve with live reload)
npm run dev

# Build for production
npm run build

# Clean build directory
npm run clean

# Build from clean state
npm run build:clean

# Serve built site
npm run serve

# Watch files for changes
npm run watch
```
